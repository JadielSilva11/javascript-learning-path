# JavaScript Learning Path

**Este repositório foi construido durante a realização do curso de JavaScript do Curso em Vídeo**

Este repositório reúne meus **estudos e práticas na linguagem JavaScript**, abordando desde os **fundamentos da linguagem** até conceitos essenciais do **desenvolvimento web** e **programação moderna**.

O objetivo é construir uma base sólida em JavaScript, tanto para aplicações front-end quanto para noções iniciais de back-end.

---

## 📚 Conteúdos Abordados

### 🔹 Fundamentos da Linguagem
- Sintaxe básica  
- Tipos de dados  
- Variáveis (`var`, `let`, `const`)  
- Operadores  
- Estruturas condicionais  
- Estruturas de repetição  
- Funções  

### 🔹 JavaScript Moderno
- Arrow functions  
- Destructuring  
- Template literals  
- Spread e rest operator  
- Módulos  

### 🔹 Manipulação do DOM
- Seleção e manipulação de elementos  
- Eventos  
- Interação com o usuário  
- Validação de formulários  

### 🔹 Tópicos Complementares
- Tratamento de erros  
- Programação assíncrona (`callbacks`, `promises`, `async/await`)  
- Consumo de APIs (fetch)  
- Introdução ao Node.js  

### 🔹 Projetos e Exercícios
- Exercícios práticos  
- Desafios de lógica  
- Pequenos projetos web
