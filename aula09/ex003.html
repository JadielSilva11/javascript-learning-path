Iniciando com o DOM.

O que é o DOM?

DOM(Document Object Model) é uma representação de tudo que existe dentro de uma página HTML.

O DOM é representado em forma de árvore e é ele quem permite que o JavaScript leia, modifique, crie e remova elementos do HTML, além de permitir que ele altere estilos, responda a eventos e cliques.

Como o DOM funciona?

O DOM é criado pelo navegador e sem ele o JavaScript não teria como mexer na página web. O Navegador transforma o HTML em objetos e quando uma página é atualizada o HTML dela não foi diretamente modificado, e sim, os objetos do DOM que foram atualizados.

Por que utilizar o DOM?

Quando utilizamos o navegador, o servidor envia um arquivo HTML junto com o código em JavaScript, Então o navegador converte os elementos HTML em objetos e cria a árvore DOM para quando a página for atualizada o HTML continua o mesmo, o que é atualizado são os objetos do DOM.

O navegador não tem permissão para modificar os arquivos HTML do site, por isso ele precisa criar uma representção dos arquivos (DOM) para pode atualizar a página.
